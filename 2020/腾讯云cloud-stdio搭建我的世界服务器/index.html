发表于 2020-02-25 |
字数统计: 771 | 阅读时长 ≈ 2

注：本教程主要面向没有任何基础的用户，部分语言可能表达不是很准确。

准备工作:
1.手，大脑(滑稽)
2.拥有一个Coding账号,因为接下来的注册cloud studio需要Coding账号

一,基础配置

  • 注册且登陆Cloud studio
  • 进入该界面
  • 新建一个工作空间并打开
  • 等待一段时间后进入这个画面
  • 按 Ctrl+ ~(键盘左上方ESC下面的按键)或点击呼出终端
  • 接下来就是软件库更新,你需要依次在终端输入以下指令：
  • 接下来需要安装Minecraft服务器运行所需要的一些基础运行环境，你需要输入以下指令： 注：若出现如图所示的情况（正常情况都会出现） 出现此语句输入“Y”即可

二,服务器搭建

  • 新建两文件夹,依次输入以下命令(括号内的内容不用输入)
  • 接着就是下载安装服务器
    打开服务器文件下载地址,这里以Mcversions 为例
    打开网址后 选择你要下载的服务器文件
    这里以1.14.3为例 单击下载后出现以下界面 将鼠标移至第一个按钮，右键复制链接地址
    接着进入刚刚新建的文件夹
    输入 cd Minecratf
    再次输入 wget 刚才复制的链接地址 等待下载完成即可，图片右下角显示的是已经下载数量，下载速度，剩余下载时间
  • 下载完成后,在终端输入以下指令
    java -Xms512M -Xmx1024M -jar server.jar nogui 在这条命令结束并返回一个错误之后，将会生成一个新的 eula.txt 文件。你需要同意那个文件里的协议。你可以通过下面这条命令将 eula=true 添加到文件中：
    sed -i.orig ‘s/eula=false/eula=true/g’ eula.txt
  • 其余就是服务器设置（自行百度）

三,端口映射

  • 现在我们来下载Sakura frp端口映射
    注册过程已跳过
    打开管理面板，如图 首先创建一条隧道 服务器自选，我的世界服务器ip和端口在服务器设置文件中可以查看(自行百度)
    完成后点击软件下载
  • 接着在终端输入wget 复制的链接地址
    下载完成后先用tar -xzvf Sakura_frpc_linux_amd64.tar.gz解压文件，文件夹下会多出来一个Sakura_frpc_linux_amd64(文件可能不同，多出来的那个就是)文件。输入命令`cd Work，直接回车，再次输入命令./Sakura_frpc_linux_amd64就ok了。
    注;如使用./打开文件显示权限不足 请使用指令 chmod 777 Sakura_frpc_linux_amd64
  • 接着就是使用映射（此部分不截图了）
    打开软件后
    输入账号 密码
    接着选择你刚刚创建的隧道的路线，即可完成映射，复制地址，开始游玩吧
