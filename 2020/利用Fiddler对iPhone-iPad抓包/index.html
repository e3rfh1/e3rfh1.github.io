利用Fiddler对iPhone/iPad抓包

下载Fiddler

Fiddler for Windows

Fiddler for Mac or Linux

注:该软件安装完成后不会自动创建快捷方式，需手动到安装目录为 Fiddler.exe 创造快捷方式

使用Fiddler

  • 设置

    打开Fiddler，进入Tools-Options。

    在”HTTPS”标签页，开启截获HTTPS的功能。

    在”Connections”标签页将端口改为8888(端口号可随意更改)，并允许其他设备连接Fiddler。

    重启 Fiddler

  • 使用

    命令行中输入ipconfig，查看IP配置，获取本机IP

    进入iOS的Wi-Fi设置，更改HTTPS代理配置。模式设为手动，服务器设为电脑的内网IP，端口设为8888，鉴定无需打开，最后保存设置。

    利用Safari，地址框输入电脑内网IP+英文冒号+端口，形如192.168.1.100:8888，点击”FiddlerRoot certificate”，安装一个名为”DO_NOT_TRUST_FiddlerRoot”的描述文件并信任。

    至此，回到PC上的Fiddler，就可以看到成功从iPhone/iPad截获的包了。
